#
# ES6 sandbox, nodejs/npm/babel/mocha
#

How to get set up
-----------------
Get node: http://nodejs.org/dist/latest/  (eg. win-x64 → just node.exe & node.lib)
Get npm: http://nodejs.org/dist/npm/ → unpack to the node directory.
Add node & npm to path
> npm install

Usage
-----
start mocha in watch mode and start exerimenting with the files in the test/ folder.
> mocha --watch
